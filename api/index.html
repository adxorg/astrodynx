Main Package#

Two-body Integrals#

astrodynx.twobody.orb_integrals.ang_momentum_vec(r, v)[source]#

Returns the angular momentum vector of a two-body system.

Parameters:

  • r ((3,) ArrayLike) – Position vector of the object.

  • v ((3,) ArrayLike) – Velocity vector of the object.

Returns:

out – Angular momentum vector of the object.

Return type:

(3,) Array

Notes

The angular momentum vector is calculated using the cross product of the position and velocity vectors:

\[ \mathbf{h} = \mathbf{r} \times \mathbf{v} \]
where \(\mathbf{h}\) is the angular momentum vector, \(\mathbf{r}\) is the position vector, \(\mathbf{v}\) is the velocity vector.

References

Battin, 1999, pp.115.

Examples

>>> import jax.numpy as jnp
>>> from astrodynx.towbody.orb_integrals import ang_momentum
>>> r = jnp.array([1.0, 0.0, 0.0])
>>> v = jnp.array([0.0, 1.0, 0.0])
>>> ang_momentum(r, v)
Array([0., 0., 1.])
astrodynx.twobody.orb_integrals.eccentricity_vec(r, v, mu)[source]#

Returns the eccentricity vector of a two-body system.

Parameters:

  • r ((3,) ArrayLike) – Position vector of the object.

  • v ((3,) ArrayLike) – Velocity vector of the object.

  • mu (DTypeLike) – Gravitational parameter of the central body.

Returns:

out – Eccentricity vector of the object.

Return type:

(3,) Array

Notes

The eccentricity vector is calculated using the formula:

\[ \mathbf{e} = \frac{\mathbf{v} \times \mathbf{h}}{\mu} - \frac{\mathbf{r}}{r} \]
where \(\mathbf{e}\) is the eccentricity vector, \(\mathbf{v}\) is the velocity vector, \(\mathbf{h}\) is the angular momentum vector, \(\mu\) is the gravitational parameter, and \(\mathbf{r}\) is the position vector.

The result is a 3D vector representing the eccentricity of the object’s orbit in the two-body system.

References

Battin, 1999, pp.115.

Examples

>>> import jax.numpy as jnp
>>> from astrodynx.towbody.orb_integrals import eccentricity_vec, ang_momentum_vec
>>> r = jnp.array([1.0, 0.0, 0.0])
>>> v = jnp.array([0.0, 1.0, 0.0])
>>> mu = 1.0
>>> h = ang_momentum_vec(r, v)
>>> eccentricity_vec(r, v, mu)
Array([-1.,  0.,  0.])
astrodynx.twobody.orb_integrals.equation_of_orbit(p, e, f)[source]#

Returns the equation of the orbit for a two-body system.

Parameters:

  • p (DTypeLike) – Semiparameter of the object’s orbit.

  • e (DTypeLike) – Eccentricity of the object’s orbit.

  • f (DTypeLike) – True anomaly of the object’s orbit.

Returns:

out – radius.

Return type:

DTypeLike

Notes

The equation of the orbit is calculated using the formula:

\[ r = \frac{p}{1 + e \cos(f)} \]
where \(r\) is the distance from the central body, \(p\) is the semiparameter, \(e\) is the eccentricity, and \(f\) is the true anomaly.

References

Battin, 1999, pp.116.

Examples

>>> import jax.numpy as jnp
>>> from astrodynx.towbody.orb_integrals import equation_of_orbit
>>> p = 1.0
>>> e = 0.5
>>> f = jnp.pi / 4  # 45 degrees in radians
>>> equation_of_orbit(p, e, f)
0.7071067811865476
astrodynx.twobody.orb_integrals.orb_period(a, mu)[source]#

Returns the period of elliptic motion.

Parameters:

  • a (DTypeLike) – Semimajor axis of the object’s orbit.

  • mu (DTypeLike) – Gravitational parameter of the central body.

Returns:

out – Period of the object’s orbit.

Return type:

DTypeLike

Notes

The period is calculated using the formula:

\[ P = 2\pi \sqrt{\frac{a^3}{\mu}} \]
where \(P\) is the period, \(a\) is the semimajor axis, and \(\mu\) is the gravitational parameter.

References

Battin, 1999, pp.119.

Examples

>>> import jax.numpy as jnp
>>> from astrodynx.towbody.orb_integrals import orb_period
>>> a = 1.0
>>> mu = 1.0
>>> orb_period(a, mu)
6.283185307179586
astrodynx.twobody.orb_integrals.semimajor_axis(r, v, mu)[source]#

Returns the semimajor axis of a two-body system.

Parameters:

  • r (DTypeLike) – Magnitude of the position vector of the object.

  • v (DTypeLike) – Magnitude of the velocity vector of the object.

  • mu (DTypeLike) – Gravitational parameter of the central body.

Returns:

out – Semimajor axis of the object’s orbit.

Return type:

DTypeLike

Notes

The semimajor axis is calculated using the formula:

\[ a = \left(\frac{2}{r} - \frac{v^2}{\mu}\right)^{-1} \]
where \(a\) is the semimajor axis, \(r\) is the magnitude of the position vector, \(v\) is the magnitude of the velocity vector, and \(\mu\) is the gravitational parameter. The result is a scalar value representing the semimajor axis of the object’s orbit in the two-body system.

References

Battin, 1999, pp.116.

Examples

>>> import jax.numpy as jnp
>>> from astrodynx.towbody.orb_integrals import semimajor_axis
>>> r = 1.0
>>> v = 1.0
>>> mu = 1.0
>>> semimajor_axis(r, v, mu)
1.0
astrodynx.twobody.orb_integrals.semiparameter(h, mu)[source]#

Returns the semiparameter of a two-body system.

Parameters:

  • h (DTypeLike) – Magnitude of the angular momentum vector of the object.

  • mu (DTypeLike) – Gravitational parameter of the central body.

Returns:

out – Semiparameter of the object’s orbit.

Return type:

DTypeLike

Notes

The semiparameter is calculated using the formula:

\[ p = \frac{h^2}{\mu} \]
where \(p\) is the semiparameter, \(h\) is the magnitude of the angular momentum vector, and \(\mu\) is the gravitational parameter. The result is a scalar value representing the semiparameter of the object’s orbit in the two-body system.

References

Battin, 1999, pp.116.

Examples

>>> import jax.numpy as jnp
>>> from astrodynx.towbody.orb_integrals import semiparameter
>>> h = 1.0
>>> mu = 1.0
>>> semiparameter(h, mu)
1.0

Kepler Equation#

astrodynx.twobody.kepler_equation.mean_anomaly_ecliptic(eccentricity, eccentric_anomaly)[source]#

Returns the mean anomaly in the ecliptic frame.

Parameters:

  • eccentricity (DTypeLike) – Eccentricity of the orbit.

  • eccentric_anomaly (DTypeLike) – Eccentric anomaly of the orbit.

Returns:

out – Mean anomaly in the ecliptic frame.

Return type:

Array

Notes

The mean anomaly is calculated using the formula:

\[ M = E - e \sin(E) \]
where \(M\) is the mean anomaly, \(E\) is the eccentric anomaly, and \(e<1\) is the eccentricity.

References

Battin, 1999, pp.160.

Examples

>>> import jax.numpy as jnp
>>> from astrodynx.twobody.kepler_equation import mean_anomaly_ecliptic
>>> mean_anomaly_ecliptic(0.1, jnp.pi/4)
Array(0.78539816)
astrodynx.twobody.kepler_equation.mean_anomaly_hyperbolic(eccentricity, hyperbolic_anomaly)[source]#

Returns the mean anomaly in the hyperbolic frame.

Parameters:

  • eccentricity (DTypeLike) – Eccentricity of the orbit.

  • hyperbolic_anomaly (DTypeLike) – Hyperbolic anomaly of the orbit.

Returns:

out – Mean anomaly in the hyperbolic frame.

Return type:

Array

Notes

The mean anomaly is calculated using the formula:

\[ M_h = e \sinh(H) - H \]
where \(M_h\) is the mean anomaly, \(H\) is the hyperbolic anomaly, and \(e>1\) is the eccentricity.

References

Battin, 1999, pp.168.

Examples

>>> import jax.numpy as jnp
>>> from astrodynx.twobody.kepler_equation import mean_anomaly_hyperbolic
>>> mean_anomaly_hyperbolic(1.1, jnp.pi/4)
Array(0.78539816)
